# Machine Learning - Retenção de Clientes (Churn Prediction)

## 📊 Descrição do Projeto
Projeto de Machine Learning para predição de churn (abandono) de clientes, utilizando técnicas de análise de dados e modelagem preditiva.

## 🎯 Objetivo
Desenvolver um modelo capaz de identificar clientes com maior probabilidade de churn, permitindo ações proativas de retenção.
